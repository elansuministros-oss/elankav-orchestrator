# VOICE-001

Estado: APROBADO

Este documento define la identidad oficial de voz de ELAN IA antes de la implementación de STT/TTS.

Principios:
- Identidad independiente del proveedor.
- Arquitectura Adapter -> Service -> Business.
- Voz masculina, natural, confiable, elegante y consistente entre español, inglés y mandarín.
- Diccionario de pronunciación extensible.
- Configuración persistente.
