# Operadores

Operadores previstos:

- ARQ: arquitectura
- QA: validación
- PDF: documentos y generación PDF
- EMC: catálogo e importaciones
- AI23: motor de costos
- WAI: WhatsApp y ELAN AI
- WEB: frontend
- API: backend
- DB: Supabase
- DEVOPS: VPS, Docker, Nginx y despliegues

Todo operador deberá:

- trabajar en rama aislada;
- registrar archivos modificados;
- ejecutar validaciones;
- devolver evidencia;
- evitar cambios destructivos;
- solicitar aprobación para producción.
